# Exam Seating Arrangement System

A web-based application developed using Java, JSP, Servlets, and MySQL to automate
the exam seating arrangement process for students in an institution.

## Features
- Student and exam data management
- Automatic seating allocation
- JSP-based user interface
- Servlet-based backend logic
- MySQL database integration

## Tech Stack
- Java
- JSP & Servlets
- MySQL
- Apache Tomcat

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Configure Apache Tomcat server
4. Import database using provided SQL script
5. Run the project on localhost
